Comandos Git:

git init (inicializa o git em um repositorio)

git status (mostra os status do repositorio)

git clone "link do repositorio para clonar" (clona repositorio do GitHub)

git add (adicionar arquivo novo)

git add * (adiciona todos os arquivos novos)

git commit -m "Mensagem" (comita repositorios)

git push origin main/master (enviar repositorio para o GitHub)
